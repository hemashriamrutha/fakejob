# Fake Job Detection System 🛡️

A Django-based web application that detects fake job postings using machine learning.

## 🚀 Features
- Detects fake job postings
- URL input for LinkedIn job listings
- Machine Learning model with imbalanced-learn support

## 🛠 Tech Stack
- Django, Python
- HTML, CSS, JavaScript
- scikit-learn, imbalanced-learn
- SQLite database

## ⚡ Installation
```bash
git clone https://github.com/hemashriamrutha/fakejob.git
cd fakejob
pip install -r requirements.txt
python manage.py runserver
